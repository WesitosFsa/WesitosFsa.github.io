Python

Python es un lenguaje de programación ampliamente utilizado en análisis de datos. Las bibliotecas clave para esto son NumPy y Pandas, que ofrecen herramientas para cálculos numéricos y manipulación de datos tabulares respectivamente. Para visualización, Matplotlib y Seaborn son útiles. Además, Python cuenta con conectores como pymongo para trabajar con bases de datos MongoDB y SQLAlchemy para bases de datos SQL, facilitando la manipulación y análisis de datos desde Python

Herramientas de Analisis

Looker Studio y Power BI son herramientas de análisis de datos. Looker Studio se centra en la exploración y visualización de datos, con énfasis en la colaboración y personalización. Mientras tanto, Power BI, de Microsoft, ofrece visualizaciones interactivas y análisis predictivo, siendo conocido por su facilidad de uso y capacidad para manejar grandes volúmenes de datos en tiempo real.

Bases de Datos

Las bases de datos NoSQL son sistemas que no solo utilizan SQL y se adaptan mejor a la escalabilidad horizontal y la flexibilidad de esquema. Son ideales para aplicaciones web de alto tráfico como redes sociales, ofreciendo velocidad y capacidad de adaptación rápida. Ejemplos incluyen MongoDB
MySQL es un sistema de gestión de bases de datos relacional que utiliza SQL para consultas y manipulación de datos. Es estable, rápido y escalable

Pandas

Es una biblioteca de Python que proporciona estructuras de datos flexibles y herramientas para manipular y analizar datos

NumPy

Es una biblioteca fundamental para el cálculo numérico en Python. Proporciona una estructura de datos de matriz

SQLAlchemy

Es una biblioteca de Python que facilita la interacción con bases de datos relacionales y nos facilita la coneccion de pyhon con BD mysql

Matplotlib

Es una biblioteca de visualización de datos en Python que permite crear una amplia variedad de gráficos y visualizaciones de alta calidad

Pymongo

Es una biblioteca de Python que permite interactuar con bases de datos

PyCharm

Es un entorno de desarrollo integrado (IDE) para Python que permite usar code de manera mas interactiva

Jupyter Notebook

Es un entorno de desarrollo interactivo basado desarrollado para Python

WebScraping

Herramientas usadas para la visualizaciones de datos

Looker Studio // Power Bi

Power Bi

Es una plataforma de análisis empresarial que permite a los usuarios visualizar y compartir datos de manera intuitiva. Es útil para crear informes interactivos, paneles y cuadros de mando a partir de diversas fuentes de datos, facilitando la toma de decisiones informadas.

Looker Studio

Es una herramienta de inteligencia empresarial que se centra en la exploración de datos y la creación de modelos de negocio personalizados. Permite a los usuarios analizar datos en tiempo real y colaborar en la toma de decisiones estratégicas basadas en información precisa
